# angular-energy-diagnostics
Directive angular pour générer les diagnostiques énergétiques (ges + dpe)

#### injection du module
````
angular.module('moduleName', ['ngEnergyDiagnotic']):
```
#### Utilisation
```
<ges ng-letter="A"></ges>
<dpe ng-value="20"></dpe>
```
